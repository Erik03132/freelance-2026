# 🕯️ РИТУАЛЫ ФРИЛАНС-АГЕНТА (Operational Protocols)

Этот документ содержит священные команды и процедуры, которые ГАРАНТИРУЮТ готовность систем перед началом охоты за проектами.

---

## 🕵️‍♂️ РИТУАЛ №1: "ПРОБУЖДЕНИЕ ШЕРЛОКА" (Pre-flight Check)
**Команда:**
```bash
cd /Users/igorvasin/freelance-2026/freelance-agent && npm run sherlock-start
```

### Зачем это нужно?
Это не просто запуск. Это **глубокая верификация** всех систем GBP (Global Boot Protocol):
1.  **Readiness Certificate**: Проверка связи с Gemini, Perplexity и Tavily.
2.  **Deduction Probe**: Тестовый запуск анализа на сложной задаче (Telegram-бот).
3.  **KB Sync**: Проверка доступности базы знаний (9 техник) и скиллов.
4.  **DB Pulse**: Проверка записи в `/tmp/freelance_active.db`.

**Когда выполнять:** Каждый раз перед началом рабочей сессии или после внесения изменений в код агента.

## ⚔️ РИТУАЛ №2: "НЕПРЕРЫВНЫЙ МОНИТОРИНГ" (24/7 Watch)
**Команда:**
```bash
cd /Users/igorvasin/freelance-2026/freelance-agent && npm run agents-start
```

### Что делает:
Запускает Sherlock в режиме **COLLECTOR-ONLY**. Агент будет обходить Kwork и FL.ru каждые 5 минут, сохранять задачи в общую базу и ждать вашего сигнала в Дашборде.
