## 5. Генерация коммерческих предложений

## Проекты/идеи:

DZone‑проект «I Built an Auto Proposal Generator LLM and Open-Sourced It» — тулза, которая на основе документации продукта + таблицы требований собирает готовое предложение в Word/Excel. dzone

GitHub: client-proposal-generator-tool — web‑приложение для генерации PDF‑коммерческих предложений из формы (бюджет, цели, сроки). github

## Что взять:

Шаблонный подход: таблица требований / поля формы → генерация структурированного предложения по шаблону (заголовки, секции, таблицы, budget, scope, timeline). dzone

Идею разделения: «данные о проекте» + «фиксированный брендированный шаблон» → PDF/Docx. Ты можешь так же хранить свои шаблоны писем и коммерческих. github
