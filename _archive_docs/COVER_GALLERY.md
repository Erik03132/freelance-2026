# 🖼️ Галерея обложек для фриланса (Kwork / Freelance.ru)

Здесь собраны сгенерированные агентом премиальные фоны для обложек кворков.
**Инструкция по использованию:**
1. Открой [Figma](https://figma.com) (или любой графический редактор).
2. Создай фрейм размером **660x440** (или `1320x880` для Retina-качества).
3. Перетащи любую из этих обложек на фрейм и растяни, чтобы она заполнила весь фон (оригиналы сгенерированы в формате квадрата 640x640, поэтому они легко кадрируются по центру).
4. Добавь крупный белый или черный текст с названием твоей услуги.
5. Экспортируй и загружай на платформу!
