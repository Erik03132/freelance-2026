## 🎯 Концепция проекта

### Основная идея
Автономный агент для поиска и отклика на фриланс-заказы **без авторизации** на биржах.

### Рабочий процесс
```
1. Агент сканирует Kwork, Freelance.ru, FL.ru (публичные данные)
   ↓
2. Оценивает задачи по навыкам и качеству ТЗ
   ↓
3. Генерирует персонализированный ответ:
   • Для лекций — план + 30% готового текста
   • Для кода — архитектурное решение + пример
   • Для текстов — стиль + пример вступления
   • Для дизайна — концепция + референсы
   • Для данных — план + JSON структура
   ↓
4. Пользователь копирует/отправляет ответ
   ↓
5. Переходит на биржу и откликается вручную
   ↓
6. Выполняет работу через Antigravity IDE
   ↓
7. Отправляет прототип заказчику
```

```sql
  platform TEXT NOT NULL,

  description TEXT,
  budget_amount REAL,
  budget_currency TEXT,
  budget_type TEXT,
  skills TEXT,              -- JSON
  client_name TEXT,

  status TEXT DEFAULT 'new',

  updated_at TEXT DEFAULT CURRENT_TIMESTAMP
```

```sql
CREATE TABLE proposals (
```

## 📊 Известные проблемы

### 1. `proposals_count` не заполняется
**Статус:** ⚠️ Не реализовано

Парсеры не извлекают количество откликов из бирж. Нужно обновить адаптеры:
- Kwork: добавить селектор для `.projects__responses`
- Freelance: добавить селектор для `.post-card__proposals`

### 2. Автоотправка не работает полностью
**Статус:** ⚠️ Частично реализовано

Функция `POST /api/auto-bid` требует:
- Скомпилированный `auto-bid.ts` в `dist/`
- Учётные данные в `.env`
- Playwright browser installed

Текущий режим: открывает браузер, но требует ручного подтверждения.

### 3. Learning Service не активируется
**Статус:** ⚠️ Требует проверки

`LearningService` сохраняется в `dist/`, но может не импортироваться в dashboard из-за путей.

**Решение:** Проверить `dashboard/src/app/api/proposals/route.ts` — динамический импорт.

### 4. Требуется API ключ Gemini для генерации ответов
**Статус:** ⚠️ Требует настройки пользователем

Без API ключа функция генерации ответов не работает.

**Решение:** 
1. Получить ключ: https://aistudio.google.com/app/apikey
2. Создать файл `dashboard/.env.local` с ключом
3. Перезапустить dashboard

**Подробности:** `dashboard/GEMINI_SETUP.md`
