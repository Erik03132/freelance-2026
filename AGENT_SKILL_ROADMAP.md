## 🎨 Рембрандт (Дизайнер)

### Текущие навыки
- UI/UX дизайн, веб-дизайн
- Генерация концептов и мудбордов
- Дизайн-токены и компонентные системы

### Потенциальные скиллы для прокачки

#### 1. Print Design (Полиграфия)
**Триггер:** Задачи на каталоги, упаковку, мерч, брендирование транспорта
**Что нужно:**
- [ ] Скилл «Print Rules» — вылеты 3 мм, CMYK, 300 dpi, шрифты в кривые
- [ ] Библиотека развёрток коробок (стандарты FEFCO → SVG-генератор)
- [ ] Библиотека лекал автомобилей (Газель, Спринтер, Ларгус)
- [ ] Интеграция с Figma API (векторный выход вместо растра)
- [ ] Конвертер RGB → CMYK PDF/X (скрипт на Python/Node)

**Оценка трудозатрат:** 2–3 недели
**Окупаемость:** 1 заказ (100–200к)

#### 2. Motion Design (Анимация)
**Триггер:** Задачи на анимированные баннеры, Lottie, видео-превью
**Что нужно:**
- [ ] Генерация Lottie JSON анимаций
- [ ] Интеграция с After Effects (через скрипты) или Rive
- [ ] Шаблоны анимированных Stories/Reels

**Оценка трудозатрат:** 1–2 недели

#### 3. 3D-визуализация
**Триггер:** Задачи на мокапы продуктов, 3D-рендеры для маркетплейсов
**Что нужно:**
- [ ] Интеграция с Blender (через Python API)
- [ ] Библиотека 3D-мокапов (коробки, футболки, кружки)

**Оценка трудозатрат:** 2–3 недели
