### Техническая реализация: `vk_market_sync.py`

Что делает скрипт:
1. Читает каталог пород из `angelochka_unified_brain.json` (SSoT)
2. Загружает фото из `vezem/dist/client/*.png` через `photos.getMarketUploadServer`
3. Создаёт карточки через `market.add` API
4. Синхронизирует цены/наличие при каждом запуске

Ключевое ограничение VK API:
```
⚠️ market.add требует User Token (не Community Token!)
   → Используем VK_USER_TOKEN из .env
   → Срок жизни: 24 часа
   → IP-привязка: тот же IP, с которого получен
```

Алгоритм синхронизации:
```
Каскад: Bitrix CRM → unified_brain.json → VK Market
1. sync_products.py  (уже есть)  → обновляет brain из Bitrix
2. vk_market_sync.py (НОВЫЙ)     → пушит brain → VK Market
3. Cron на VPS: раз в день
```

## 🔵 ФАЗА 4 — VK Mini App: ВезёмЦыплят Заказ

### Как Mini App увязывается с сообществом

```
ВК Сообщество → Кнопка «🛒 Заказать» в меню → Mini App
Карточка товара → кнопка «Купить» → Mini App
Пост на стене → CTA кнопка → Mini App
Анжела в чате → ссылка на Mini App → Mini App

Mini App → Каталог (данные из VK Market) → Форма заявки (VK ID) → Bitrix24
```

### Зачем Mini App если уже есть сайт vezemcip.ru?

| Критерий | Сайт vezemcip.ru | VK Mini App |
|----------|-----------------|-------------|
| Авторизация | Нет (анонимно) | ✅ VK ID автоматически |
| Контакт клиента | Нужно заполнять форму | ✅ Имя + телефон из ВК |
| Уведомления | Нет | ✅ Push через ВК |
| Виральность | Нет | ✅ «Поделиться» внутри ВК |
| Трение при заказе | Высокое (новый сайт) | Минимальное (внутри ВК) |
| Данные из VK Market | Нет | ✅ Единый каталог |
| Анжела AI | ✅ Виджет на сайте | ✅ Кнопка → чат сообщества |

### Технический стек Mini App

```
Frontend:  React + VKUI (официальный UI-kit ВКонтакте)
Scaffold:  npx @vkontakte/create-vk-mini-app (шаблон VKUI+Bridge+Router)
Bridge:    @vkontakte/vk-bridge (авторизация, уведомления, шеринг)
Deploy:    @vkontakte/vk-miniapps-deploy → VK CDN (бесплатно!)
Backend:   Наш VPS (72.56.38.19) → FastAPI endpoint /api/vk-order
CRM:       Bitrix24 webhook (уже настроен!)
Каталог:   VK API market.get → читаем из VK-магазина (Фаза 2)
```
