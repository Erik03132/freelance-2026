## Кеш вопросов-ответов (Telegram + Avito + Телефон)

### Проблема
LLM-ответ занимает 2-40 секунд. Клиент ждёт.

### Решение
1. Собрать **все диалоги** из:
   - Расшифровок телефонных звонков
   - Сообщений Avito (чат)
   - Telegram-бота Заботкиной
2. Выделить **повторяющиеся вопросы** (LLM-кластеризация)
3. Сгенерировать **один идеальный ответ** на каждый вопрос
4. Сохранить в JSON-кеш: `{question_hash: {answer, audio_path}}`
5. При живом звонке:
   - Расшифровка → хеш вопроса → мгновенный ответ из кеша (< 1 сек)
   - Если вопрос новый → LLM → сохранить в кеш

### Этапы
1. Собрать датасет диалогов (телефон + Avito + TG)
2. Кластеризовать вопросы (embeddings + clustering)
3. Сгенерировать ответы (LLM + Kore TTS)
4. Сохранить в `qa_cache.json`
5. Интегрировать в angela_outbound_v2.py
