# Auto Reels Maker
Умный генератор коротких видео (Reels / Shorts) из массива фото и видео под музыкальный трек.

## Возможности
- Анализ аудио и расчет необходимого количества фото и видео для покрытия всей длины трека.
- Автоматический подгон фото под формат 9:16 (умный кроп / размытый фон).
- Автоматическая нарезка видео-вставок (выбор сочных фрагментов).
- Наложение плавных переходов (Crossfade) и эффектов наезда камеры (Кен Бернс).

## Требования (установка)
```bash
pip install moviepy librosa Pillow opencv-python numpy
```

## Как использовать
1. Сложите все фото и видео в папку `media`.
2. Положите музыкальный трек `track.mp3` в корень.
3. Запустите `python auto_reels.py`
