# Legal Graph RAG — Навигация по нормативной базе - [PROJECT: ai-grant-consalt]

## Goal
Обеспечить точный поиск по законодательной базе с учётом связей между документами (кросс-ссылки, зависимости, стоп-факторы).

## Когда активировать
- Клиент задаёт вопрос о конкретном законе/статье
- Нужно проследить цепочку связей (например, ИТ-аккредитация → льготы → ОКВЭД)
- Подготовка материалов для кейсов и статей
- Проверка стоп-факторов перед подачей на грант

## 🗺️ Граф связей (основные кластеры)

### Кластер 1: ИТ-льготы
```
ПП РФ №1729 (критерии аккредитации)
  ├── ОКВЭД 62.01 (профильный код)
  ├── п. 1.15 ст. 284 НК РФ (5% на прибыль)
  ├── ст. 427 НК РФ (7.6% взносы)
  └── Аккредитация Минцифры (Приказ №643)
        └── через Госуслуги (15 рабочих дней)
```

### Кластер 2: Гранты ФСИ
```
Старт-1 (до 5 млн)
  ├── Стоп-факторы (задолженность, ОКВЭД, ИС)
  ├── Scoring (новизна, коммерциализация, команда)
  ├── Обязательства (НИОКР, РИД, сайт)
  └── Регистрация РИД (Роспатент / ФИПС)
        └── Для ПО: ~5000₽, 2-3 мес
```

### Кластер 3: Банкротство
```
ФЗ-127 «О несостоятельности»
  ├── Физлица (от 500K₽ + 3 мес)
  │     └── Внесудебное через МФЦ (25K - 1M₽)
  ├── Юрлица (4 стадии)
  │     └── Субсидиарка (ст. 61.11-61.12)
  │           └── ст. 9 ФЗ-127 (обязанность подать в 1 мес)
  └── Ликвидация ООО (ГК РФ, ст. 61-64)
```

### Кластер 4: Крупные гранты
```
РФРИТ (20-500 млн) → аккредитация + импортозамещение
Сколково (ФЗ-244) → 0% прибыль + 0% НДС + микрогранты
```

### Кластер 5: Агро-гранты
```
Агростартап (до 7 млн) → КФХ и ИП
Агротуризм (до 10 млн) → село + туризм
```

### По типу вопроса

|---------------|---------------|---------|

| «Можно ли получить грант?» | fsi_start_1 | → fsi_stop_factors → okved_62_01 |

| «Как списать долги?» | bankruptcy_physical | → bankruptcy_legal → liquidation |

### Правило 3 хопов
При ответе на вопрос — всегда проверяй **3 связанных узла** для полного контекста.
