# Frontend Technical Standards — AI Grant Consult

Этот документ определяет «золотые стандарты» разработки фронтенда для всех агентов и разработчиков в экосистеме. Мы ориентируемся на уровень Senior-разработки (на базе Hack Frontend Roadmap) и требования поисковых систем 2026 года (SXO/GEO).

---

## 🚀 1. Performance & Core Web Vitals (Цели)
Каждая страница должна соответствовать метрикам «Зеленой зоны» Google Lighthouse:
- **LCP (Largest Contentful Paint):** < 2.5с (Цель: 1.2с)
- **INP (Interaction to Next Paint):** < 200мс
- **CLS (Cumulative Layout Shift):** < 0.1
- **Оптимизация изображений:** Только форматы WebP/AVIF. Обязательные атрибуты `loading="lazy"` (кроме первого экрана) и явные `width`/`height`.

## 🏗️ 2. Архитектура и Код
- **Модульность:** Компоненты должны быть атомарными. Логика API выносится в отдельные хуки/сервисы.
- **TypeScript:** Строгая типизация обязательна. Никаких `any`.
- **CSS:** Использование CSS Variables для дизайн-системы. Приоритет — Vanilla CSS или Tailwind (если запрошено), но с соблюдением DRY.
- **Чистота:** Удаление неиспользуемого кода (Tree Shaking), минимизация бандлов.

## 🤖 3. SXO & GEO (AI-Ready Markup)
Сайт должен быть «понятным» для Google MUM и Yandex YATI 2.0:
- **Семантика:** Строгое использование HTML5 тегов (`<main>`, `<article>`, `<section>`, `<aside>`).
- **Микроразметка:** JSON-LD для каждого типа контента (Organization, FAQ, Breadcrumbs, Product/Service).
- **Иерархия заголовков:** Только один `<h1>` на страницу. Иерархия H1 -> H2 -> H3 без пропусков.
- **Адаптивность:** Mobile-First — это база. Интерфейс должен быть удобен для управления «одним пальцем».

## ♿ 4. Доступность (A11y)
- **ARIA:** Использование атрибутов `aria-label`, `role` там, где семантики недостаточно.
- **Контрастность:** Соблюдение стандартов WCAG AA.
- **Клавиатура:** Полная навигация по сайту без мыши.

## 🎨 5. Premium Design Guidelines
- **Micro-interactions:** Плавные переходы (0.2s - 0.3s), hover-эффекты на кнопках и ссылках.
- **Typography:** Использование системных шрифтов или Google Fonts с правильным весом (font-weight).
- **Glassmorphism & Gradients:** Использование современных эффектов размытия и мягких градиентов для ощущения «дорогого» продукта.
