## Правила размещения

- **Глобальное** (используется в 2+ проектах) → `tools/` или корень
- **Специфичное для проекта** → внутрь папки проекта
- **Эксперимент/прототип** → `agent-lab/`
- **Ключи** → `ai-eggs/.env` (мастер), дублируются в проекты по необходимости
