## 🛠 Технические решения и код

### Задача 1: Gemini TTS (голос Kore)
- **Проблема**: Нужен естественный женский голос для автообзвона
- **Решение**: `gemini-2.5-flash-preview-tts`, голос Kore, US SOCKS5 прокси
- **Результат**: ✅ Голос записан, конвертирован 24kHz→8kHz, закэширован

### Задача 2: Автодозвон через Mango Office
- **Проблема**: Нужно автоматически звонить клиентам и проигрывать голосовое
- **Решение**: Mango Callback API (ext 22) → VPS baresip (auto-answer, aufile)
- **Грабли**: baresip начинает играть сразу (7с тишины в начале WAV), requests таймаутит (curl)
- **Результат**: ✅ Клиент слышит полное сообщение Kore

### Задача 3: DTMF перехват (кнопки 1/0)
- **Проблема**: Клиент нажимает 1 (да) или 0 (нет) — нужно ловить
- **Решение**: `dtmf_monitor.py` парсит screen baresip → `dtmf_handler.py` (HTTP 8086)
- **Грабли**: baresip v1.0 не имеет HTTP модуля для DTMF, screen обрезает строки
- **Результат**: ✅ Нажатие 1 → ПОДТВЕРЖДЕНО, нажатие 0 → ОТКЛОНЕНО

### Задача 4: STT — голосовое ДА/НЕТ
- **Проблема**: Не все клиенты нажимают кнопки — говорят "да" или "нет"
- **Решение**: faster-whisper (tiny, CPU) на VPS, sndfile записывает dec.wav (голос клиента)
- **Грабли**: Сначала анализировал enc.wav (наш голос!) → переключил на dec.wav
- **Оптимизация**: БИП (800Hz) после Kore как сигнал "говорите" + 10с тишины для ответа
- **Результат**: ✅ STT распознал "Капа, горы, да?" → классификатор нашёл "да" → ПОДТВЕРЖДЕНО

### Задача 5: Умный каскад LLM
- **Проблема**: Все запросы шли на дорогую модель
- **Решение**: 3 тира по сложности:
  - LITE: DeepSeek V4 Flash ($0.11)
  - STD: DeepSeek V4 Pro ($0.43)
  - PRO: Kimi K2.6 ($0.73)
- **Результат**: ✅ Экономия ~57%, маршрутизация по ключевым словам
