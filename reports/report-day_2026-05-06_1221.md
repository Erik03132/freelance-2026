### Задача 1: Ночной аудит — починка
- **Проблема**: Cron в 02:00 не сработал (Mac спал), скрипт содержал 3 бага
- **Решение**: 
  - Баг 1: `grep -c '.'` → `wc -l | tr -d ' '` (multiline output ломал `[ -eq 0 ]`)
  - Баг 2: `set -euo pipefail` → `set -eu` (SIGPIPE exit 141 при `ruff | head`)
  - Баг 3: curl с мёртвым прокси → `--connect-timeout 5` + fallback без прокси
  - Каскад: Mac 02:00 + VPS 02:05 (lock-файл координации)
- **Результат**: ✅ Оба аудитора работают, TG отправляется
