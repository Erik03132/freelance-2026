## 🎯 Ключевые идеи

### 1. Минимальный стек (1 файл vs LangChain)
- **Суть:** FastAPI + httpx.post() вместо LangChain + векторной базы + RAG
- **Ценность:** 🔴 ВЫСОКАЯ
- **Применимо:** ✅ Angela (ai-eggs), ContentCombine pipeline
- **Почему ценно:** Подтверждает наш подход - мы уже делаем так!

### 2. Системный промпт вместо RAG
- **Суть:** Если контент помещается в SYSTEM (~2500-5500 токенов) - не нужен RAG
- **Ценность:** 🔴 ВЫСОКАЯ
- **Применимо:** ✅ Angela, sherl-research
- **Почему ценно:** RAG добавляет задержку, стоимость, сложность

### 3. FAQ-эталоны вместо чёрного списка запретов
- **Суть:** 10-15 эталонных ответов работают лучше, чем 50 запретов
- **Ценность:** 🔴 ВЫСОКАЯ
- **Применимо:** ✅ Angela (ai-eggs), ContentCombine
- **Почему ценно:** Модели лучше имитируют положительные примеры

### 4. Pexels вместо генерации картинок
- **Суть:** Живое фото > AI-генерация для интерьеров, фуд-фото, моды
- **Ценность:** 🟡 СРЕДНЯЯ
- **Применимо:** ✅ ContentCombine (визуалы в Telegram)
- **Почему ценно:** Дешевле, быстрее, правдоподобнее

### 5. Чипсы вместо текстовых полей
- **Суть:** Кнопки с предзаготовленными вариантами вместо textarea
- **Ценность:** 🟡 СРЕДНЯЯ
- **Применимо:** ✅ ai-scout (формы на сайте)
- **Почему ценно:** Заполняемость формы выросла в 2.5-3 раза

### 6. Prompt caching у Anthropic
- **Суть:** Кэширование SYSTEM-промпта → экономия ~90% на input-токенах
- **Ценность:** 🟡 СРЕДНЯЯ
- **Применимо:** ✅ Angela (ai-eggs)
- **Почему ценно:** Экономия ~10x на стоимости API

### 7. Не тащить фреймворки раньше времени
- **Суть:** "Минимально достаточный" подход
- **Ценность:** 🔴 ВЫСОКАЯ

```python
# В telegram_exporter.py
import httpx

async def fetch_pexels_photo(query: str) -> str:
    """Fetch top photo from Pexels"""
    async with httpx.AsyncClient() as client:
        resp = await client.get(
            f"https://api.pexels.com/v1/search?query={query}&per_page=1",
            headers={"Authorization": PEXELS_API_KEY}
        )
        data = resp.json()
        return data["photos"][0]["src"]["large"] if data["photos"] else None
```
