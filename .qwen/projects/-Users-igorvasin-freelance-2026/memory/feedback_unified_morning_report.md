**Правило:** Один большой утренний отчёт Игорю (176203333) в 08:00 MSK.

**Why:** 14.05.2026 — чтобы не запутаться в множестве отчётов (CRM, call_quality, night_audit), все данные объединены в один.

**Структура отчёта:**
```
🌞 УТРЕННИЙ ОТЧЁТ — ДД.ММ.ГГГГ ЧЧ:ММ
Данные за ДД.ММ.ГГГГ

💰 CRM ЗА ВЧЕРА
• Сделок: N (на X₽)
• Звонков: N
• Менеджеры: топ-3 по сумме

📞 ТРАНСКРИБАЦИЯ (N звонков)
🏆 Топ пород: бройлер, мулард, агидель...
💰 Крупные заказы: ~X₽+
📈 Качество менеджеров: назвались N%, город N%, апселл N%
⚠️ Проблемы: клиент обещал перезвонить, отказы...

🌙 НОЧНОЙ АУДИТ (02:00)
• Ruff ошибок: N
• Критических: N

🚀 План на сегодня
```

**Расписание:**
- 02:00 MSK — ночной аудит кода
- 03:00 MSK — транскрибация звонков за вчера
- 08:00 MSK — **ОБЪЕДИНЁННЫЙ ОТЧЁТ** Игорю
