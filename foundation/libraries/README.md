# Foundation / Libraries

Здесь хранятся общие библиотеки, утилиты и middleware, которые используются в нескольких проектах:

- `agentClient.js` / `agentClient.py` — единая точка вызова агентов
- `skillLoader.*` — авто-регистрация скиллов
- `eventBus.*` / `logAgent.*` — логирование и очередь событий
- `security/` — общие security-хелперы

Правило: если код используется в 2+ проектах — он должен быть здесь, а не копироваться.
