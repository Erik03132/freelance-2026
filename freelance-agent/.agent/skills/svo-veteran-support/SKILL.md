## 💰 Грантовый модуль (Veteran Grant Writer)

### Программы поддержки (актуально 2025-2026)
| Программа | Сумма | Условия |
|-----------|-------|---------|
| Региональный грант для ветеранов | 100-750 тыс. руб. | ООО с ветераном-учредителем |
| Социальный контракт (соцконтракт) | до 350 тыс. руб. | ИП или самозанятые |
| Агромотиватор | 5-7 млн руб. | Агропредприятия |
| Фонд содействия инновациям | до 4 млн руб. | Технологические стартапы |

### Требования к заявке
- Удостоверение ветерана боевых действий (обязательно!)
- ООО: единственный учредитель-ветеран (или 51%+ доля)
- ОКВЭД коды — выбирать грант-совместимые
- Устав с социальной миссией (реабилитация, трудоустройство ветеранов)
- Соофинансирование: 25% от суммы гранта собственными средствами
- Регистрация в регионе подачи (прописка!)

### Чеклист документов (Московская область)
- [ ] Удостоверение ветерана боевых действий
- [ ] Паспорт РФ
- [ ] Свидетельство о регистрации ООО/ИП
- [ ] Устав ООО с социальной миссией
- [ ] Бизнес-план (по форме «Мой бизнес»)
- [ ] Выписка из ЕГРЮЛ (не старше 30 дней)
- [ ] Справка об отсутствии задолженности по налогам
- [ ] Подтверждение соофинансирования (выписка из банка)

### Формула социальной миссии
```
«[Название компании] занимается [деятельность], 
создавая рабочие места для ветеранов СВО и 
обеспечивая их социальную реабилитацию через 
[конкретные механизмы]. Проект направлен на 
[измеримый социальный результат].»
```

## 🧠 Психологический модуль (Adaptation Psychologist)

### ВАЖНО — Safety First
Если пользователь выражает суицидальные мысли:
```
НЕМЕДЛЕННО дать контакты:
- Телефон доверия: 8-800-2000-122 (бесплатно)
- Кризисная линия МЧС: 112
- Скорая психиатрическая помощь: 03
```

### Нормализующие скрипты (для бота)
```
Тревога/гипербдительность:
"То, что вы описываете — это нормальная реакция 
нервной системы на экстремальный опыт. Тело 
продолжает защищать вас, даже когда опасность миновала."

Проблемы со сном:
"Многие ветераны сталкиваются с этим в первые 
месяцы. Это не слабость — это адаптация. 
Давайте попробуем технику 4-7-8..."
```

### Быстрые техники (для бота)
```
Квадратное дыхание (Box Breathing):
1. Вдох 4 секунды
2. Задержка 4 секунды
3. Выдох 4 секунды
4. Задержка 4 секунды
→ Повторить 4 раза

Заземление 5-4-3-2-1:
Назови 5 вещей которые видишь
Назови 4 вещи которые можешь потрогать
Назови 3 звука которые слышишь
Назови 2 запаха
Назови 1 вкус
```

### Когда направлять к специалисту
- ПТСР симптомы > 4 недель
- Тяжёлая депрессия (нет сил, нет смысла)
- Употребление алкоголя/ПАВ как «лекарство»
- Вспышки агрессии с причинением вреда

## ⚖️ Правовой модуль

### Ключевые законы (часто нужны ссылки)
- **ФЗ-7** «О ветеранах» — базовые льготы
- **ФЗ-190** «О статусе военнослужащих» — выплаты
- **ПП РФ №1093** — региональные гранты
- **ФЗ-208** «Об обществах с ограниченной ответственностью»

### Типичные вопросы и ответы
```
Q: Когда можно подать на грант?
A: В среднем через 3-6 месяцев после регистрации ООО.
   Конкурсы проходят 2-3 раза в год (обычно февраль, июнь, октябрь).

Q: Нужно ли ИП для гранта?
A: Для региональных грантов лучше ООО (выше суммы).
   Для соцконтракта — ИП или самозанятый достаточно.
```
