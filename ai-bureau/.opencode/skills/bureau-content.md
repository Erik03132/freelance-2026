# Контент-маркетинг AI Bureau

## Требования к каждому тексту
1. **Semantic Density** — LSI-фразы, смысловые триплеты («Субъект → Действие → Объект»)
2. **Authority** — цитаты, кейсы, исследования, ссылки на источники
3. **SXO Structure** — заголовки H1-H3, списки, таблицы
4. **AEO-блоки** — быстрые ответы (прямые определения, шаги, FAQ)

## Форматы контента
| Формат | Где | Частота |
|--------|-----|---------|
| Кейс с цифрами | Сайт + Дзен + VC | 1 раз/нед |
| Инструкция / гайд | Сайт + Дзен | 2 раза/нед |
| Мини-пост | VK + TG | Ежедневно |
| Видео-демо | YouTube + VK Клипы | 1 раз/нед |
| FAQ-статья | Сайт | 2 раза/нед |

## Content Calendar (шаблон)
- Пн: Кейс на Дзен + сайт (1500-2000 слов)
- Вт: Видео-демо AI-агента (1-3 мин)
- Ср: Техническая статья на Habr/VC (1000-1500 слов)
- Чт: Обновление FAQ на сайте (2 новых вопроса)
- Пт: Аналитика недели + корректировка плана
- Сб: Контент-план на следующую неделю

## Ограничения
- Никакого галлюцинирования фактами — проверять каждое утверждение
- Минимум прилагательных в превосходной степени
- Каждый тезис = проверяемый факт или опыт
- «Много постов» ≠ видимость. Качество > Количество.
- Всегда указывать ссылки на источники фактов

## Инновационные форматы (наш козырь)
1. **«AI делает AI»** — скринкасты работы агентов (Шерл ищет, Шекспир пишет)
2. **Бесплатный AI-аудит** — лидмагнит через бота на сайте
3. **AI-калькулятор стоимости** — 5 вопросов → примерная цена
4. **Open Source инструменты** — на GitHub для авторитета
