## Каскадная система выбора моделей

## Принцип работы

OpenCode не имеет встроенного авто-роутинга, но реализует каскад через:

1. **Классификация задачи** — по ключевым словам и оценке рисков
2. **Переключение модели** — через сабагентов для Tier 3/4, через `small_model` для Tier 1
3. **Эскалация при неудаче** — при провале нижнего тира → переход наверх

## Каскадная система (4 уровня)

### Tier 1 — Простые задачи ($0)
**Модель:** `opencode/deepseek-v4-flash-free` (small_model)
**Ожидаемая доля:** 35-40% задач
**Макс. попыток:** 2

Используй `small_model` для:
- Исправление синтаксических ошибок и опечаток
- Форматирование кода, линтинг
- Простой рефакторинг (один файл)
- Добавление комментариев и документации
- Написание простых unit-тестов
- Переименование переменных/функций
- Чтение и объяснение кода

**КАК:** не делегируй сабагенту — small_model активируется автоматически для быстрых операций.

**Эскалация:** после 2 неудачных попыток → Tier 2.

---

### Tier 2 — Стандартные задачи ($1.74/$3.48 per 1M токенов)
**Модель:** `openrouter/deepseek-v4-pro` (основная модель)
**Ожидаемая доля:** 40-45% задач
**Макс. попыток:** 3-4

Основная рабочая лошадка. Используй для:
- Сложные баги (race conditions, async-проблемы)
- Мультифайловый рефакторинг (2-5 файлов)
- Интеграция API и HTTP-запросы
- Оптимизация производительности
- Оптимизация запросов к БД
- Улучшение обработки ошибок
- Добавление новых фич (средняя сложность)
- Написание тестов (интеграционные, e2e)

**КАК:** это модель по умолчанию — выполняй задачу сам.

**Эскалация:** после 3-4 неудач → Tier 3.

---

### Tier 3 — Продвинутые задачи ($0.30/$1.20 per 1M токенов)
**Модель:** `opencode/minimax-m2-7`
**Ожидаемая доля:** 10-15% задач
**Макс. попыток:** 2

Делегируй сабагенту `advanced`. Используй для:
- Архитектурные решения и паттерны проектирования
- Рефакторинг легаси-кода без документации
- Реализация сложных алгоритмов
- Креативное решение краевых случаев (edge cases)
- Дизайн системы и архитектура
- Нестандартные интеграции API
- Задачи с высокой неопределённостью

**КАК:** используй Task tool с `subagent_type: "advanced"`.

**Эскалация:** после 2 неудач → Tier 4.

---

### Tier 4 — Критические задачи ($5/$25 per 1M токенов)
**Модель:** `opencode/claude-opus-4-7`
**Ожидаемая доля:** <5% задач
**Макс. попыток:** 1

Делегируй сабагенту `critical`. Используй для:
- Production-critical баги (даунтайм, потеря данных)
- Платёжные системы, безопасность, аутентификация
- Когда все предыдущие тиры провалились
- Системные архитектурные изменения
- Задачи, где цена ошибки > $50
- Утечки данных, уязвимости
- Миграции БД в продакшене

**КАК:** используй Task tool с `subagent_type: "critical"`.

## Правила эскалации

### Автоматические триггеры эскалации
| Тир | После попыток → | Переход на |
|-----|-----------------|------------|
| Tier 1 | 2 неудачных | Tier 2 |
| Tier 2 | 3-4 неудачных | Tier 3 |
| Tier 3 | 2 неудачных | Tier 4 |

### Ключевые слова для пропуска тиров
- **"production", "critical", "urgent", "срочно", "критично", "продакшен"** → Tier 3 или 4
- **"simple", "quick", "formatting", "просто", "быстро", "форматирование"** → Tier 1
- **"refactor", "optimize", "bug", "рефакторинг", "оптимизация", "баг"** → Tier 2

### Эскалация по стоимости ошибки
- Цена ошибки > **$50** → сразу Tier 4 (critical)
- Цена ошибки **$10-50** → Tier 3 (advanced)
- Цена ошибки < **$10** → Tier 1 или 2

## Ручные команды

Пользователь может явно указать тир:
- **`/simple`** → Tier 1 (DeepSeek Flash Free)
- **`/standard`** → Tier 2 (DeepSeek V4 Pro) — по умолчанию
- **`/advanced`** → Tier 3 (MiniMax M2)
- **`/critical`** → Tier 4 (Claude Opus 4)
- **`/auto`** → Авто-выбор (по умолчанию)

## Учёт использования

При выборе модели логируй (мысленно):
1. Какой тир выбран и почему
2. Успех/провал задачи
3. Оценка стоимости

Целевая стоимость: **$15-20/мес**

| Тир | Доля | Модель | $/мес |
|-----|------|--------|-------|
| Tier 1 | 35-40% | DeepSeek Flash Free | $0 |
| Tier 2 | 40-45% | DeepSeek V4 Pro | $6-8 |
| Tier 3 | 10-15% | MiniMax M2 | $1-2 |
| Tier 4 | <5% | Claude Opus 4 | $7-10 |

Лимит бюджета: **$20/мес**. При приближении к лимиту — предупреждай пользователя.
