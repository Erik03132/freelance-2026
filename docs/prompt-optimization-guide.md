# Prompt Optimization Guide (Based on Article 1053516)

## Проблема

Раздутые системные промпты вызывают галлюцинации:
- Lost in the Middle (30-40% потеря внимания)
- Contextual Distraction (до 80% падение точности)
- Few-Shot Overrated (примеры не учат)

## Решение

### 1. Убрать примеры

**Было:**
```
Пример хорошего письма:
Привет! Заинтересовала вакансия...
```

**Стало:**
```
Сопроводительное письмо. 3 абзаца, русский.
```

### 2. Убрать повторы

**Было:**
```
Используй инструменты.
Всегда вызывай инструменты.
Не используй shell, используй инструменты.
```

**Стало:**
```
Используй инструменты.
```

### 3. Убрать нерелевантные термины

**Было:**
```
Ты - бэкенд разработчик. Используй Python, FastAPI, Docker,
React, Vue, TypeScript, PostgreSQL, MongoDB, Redis, Kafka...
```

**Стало:**
```
Ты - бэкенд разработчик на Python.
```

### 4. Сократить системный промпт

**Цель:** < 1000 токенов (500 оптимально)

**Текущий размер:**
- Angela: ~2000 токенов
- HH.ru bot: ~1500 токенов
- CLAUDE.md: ~800 токенов

**Целевой размер:**
- Angela: ~500 токенов
- HH.ru bot: ~400 токенов
- CLAUDE.md: ~300 токенов

## Шаблоны промптов

### Router (Angela)
```python
# Было (500 токенов)
ROUTER_PROMPT = """
Ты - роутер Angela. Твоя задача - определить роль собеседника, тему разговора и сложность запроса.

Роли: sales, support, tech, billing
Темы: pricing, features, demo, complaint, integration
Сложность: low, medium, high

Формат вывода: JSON
{
    "role": "sales",
    "topic": "pricing",
    "complexity": "medium",
    "confidence": 0.95
}

Примеры:
Вопрос: "Сколько стоит?"
Ответ: {"role": "sales", "topic": "pricing", "complexity": "low"}

Вопрос: "Как интегрировать API?"
Ответ: {"role": "tech", "topic": "integration", "complexity": "high"}
"""

# Стало (100 токенов)
ROUTER_PROMPT = """
Определи роль, тему, сложность. Формат: JSON.
Роли: sales, support, tech, billing.
"""
```

### Generator (Angela)
```python
# Было (800 токенов)
GENERATOR_PROMPT = """
Ты - менеджер Angela. Генерируй ответы на основе контекста.

Стиль: профессиональный, дружелюбный
Длина: 2-3 предложения
Язык: русский

Правила:
1. Не используй технический жаргон
2. Ссылайся на FAQ если есть
3. Предлагай следующий шаг

Примеры:
Вопрос: "Сколько стоит?"
Ответ: "Наш тариф Pro стоит 5000 руб/мес. Включает все функции. Попробуете бесплатно?"

Вопрос: "Как настроить?"
Ответ: "Настройка занимает 5 минут. Вот инструкция: [ссылка]. Нужна помощь?"
"""

# Стало (150 токенов)
GENERATOR_PROMPT = """
Генерируй ответы. Стиль: профессиональный. Длина: 2-3 предложения. Язык: русский.
"""
```

### Cover Letter (HH.ru)
```python
# Было (600 токенов)
COVER_LETTER_PROMPT = """
Напиши сопроводительное письмо.

Мой профиль: {resume_summary}
Вакансия: {vacancy_title}
Описание: {vacancy_description}

Правила:
1. ПИСАТЬ ТОЛЬКО НА РУССКОМ
2. 3-4 абзаца
3. Подпись: "С уважением, [Имя]"

Пример:
Привет! Заинтересовала вакансия {vacancy_title}.
Я программист с опытом Python. Готов решать сложные задачи.
Мой стек: Python, FastAPI, PostgreSQL.
Буду рад обсудить детали.
С уважением, [Имя]
"""

# Стало (150 токенов)
COVER_LETTER_PROMPT = """
Сопроводительное письмо. Профиль: {resume_summary}. Вакансия: {vacancy_title}.
3 абзаца, русский, подпись "С уважением, [Имя]".
"""
```

## Тестирование

### Метрики для сравнения

| Метрика | До | После | Цель |
|---------|-----|-------|------|
| Токены в промпте | 2000 | 500 | -75% |
| Галлюцинации | 30% | 10% | -67% |
| Уточняющие вопросы | 1/5 | 4/5 | +300% |
| Качество ответов | 70% | 85% | +21% |

### Тестовый сценарий

```python
# Тест Angela
test_cases = [

]
```

## Чек-лист перед отправкой промпта

- [ ] Убраны все примеры
- [ ] Убраны повторы
- [ ] Убраны нерелевантные термины
- [ ] Размер < 500 токенов
- [ ] Нет противоречивых инструкций
- [ ] Формат вывода указан явно
