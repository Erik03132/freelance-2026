## Этап 1: Сбор данных

### Источники записей
- [ ] Mango API: `GET /vpbx/api/call_history/{call_id}/recording`
- [ ] VPS: `/root/antigravity/recordings/`
- [ ] Локально: `data/recordings/`

### Расшифровка
```bash
# Скачать записи с VPS
scp root@72.56.38.19:/root/antigravity/recordings/*.wav ./recordings/

# Расшифровать через faster-whisper
python3 scripts/transcribe.py --input ./recordings/ --output ./transcripts/
```
