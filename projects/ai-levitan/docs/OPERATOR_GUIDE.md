# Инструкция по автодозвону — Levitan Dialer Bot

## Что делает бот

Бот автоматически набирает номера из базы контактов. Ты только поднимаешь трубку в Zoiper и разговариваешь с клиентом. После звонка — нажимаешь кнопку «Следующий», бот набирает следующий номер.

## 1. Откуда берутся номера

Есть **общая база** на 11 240 контактов — сельхозпроизводители России. База отфильтрована автоматически по ключевым словам:

| Категория | Что ищем |
|-----------|----------|
| Зерновые | пшеница, ячмень, кукуруза, озимая, яровая |
| Масличные | подсолнечник, рапс, соя |
| Бобовые | горох, нут, чечевица |

**Исключены:** КРС, молочное, мясное, свиньи, птица, рыба, овощи, картофель, техника, ремонт, хранение.

Бот также не звонит повторно тем, кто уже есть в результатах предыдущих дней.

## 2. Своя база контактов (необязательно)

Если у тебя свой список номеров — отправь файл прямо в Telegram боту:

- **CSV** — таблица со столбцом `Телефоны` (остальные столбцы опциональны)
- **XLS / XLSX** — Excel, бот сам сконвертирует в CSV
- **TXT** — просто список номеров, по одному на строке

Пример TXT-файла:
```
79001234567
79161234567
79087654321
```

После загрузки бот покажет: «Загружено 150 контактов. Нажми ▶ Начать обзвон».

**Важно:** для своих списков бот НЕ применяет фильтр по культурам — звонит по ВСЕМ номерам из файла.

## 3. Запуск обзвона

1. Открой Telegram → **@levitan_dialer_bot**
2. Укажи свой внутренний номер: `/ext 100`
3. Нажми кнопку **▶ Начать обзвон**
4. Бот сообщит: «Обзвон запущен! Контактов: 11 240. Сейчас набираю первый номер...»

## 4. Звонок

1. **Zoiper на телефоне звонит** → подними трубку
2. Слышишь гудки дозвона (или «абонент недоступен»)
3. Клиент отвечает: говори по скрипту
4. Завершил разговор → **положи трубку**

## 5. Следующий звонок

Нажми кнопку **⏭ Следующий**. Бот наберёт следующий номер из базы.

Повторяй: поднял → поговорил → положил → «Следующий».

## 6. Кнопки

| Кнопка | Когда нажимать |
|--------|---------------|
| ▶ Начать обзвон | В начале работы |
| ⏭ Следующий | После каждого звонка |
| ⏩ Пропустить | Пропустить контакт без звонка |
| ⏹ Стоп | Закончить работу |
| 📊 Статус | Посмотреть статистику |

## 7. После звонка (автоматически)

Бот в фоне сам сделает:
- Скачает запись разговора
- Расшифрует через ИИ (речь → текст)
- Извлечёт: имя клиента, культуру, объём, цену, статус
- Сохранит в CRM
- Пришлёт тебе карточку с результатом через 30-90 секунд

## 8. Важно

- **Команды боту — только между звонками** (когда не говоришь по Zoiper)
- Если клиент не ответил — просто жми «Следующий», бот учтёт это как «не взял»
- Твой внутренний номер (ext) сохраняется, повторно вводить не нужно
- В выходные бот автоматически фильтрует ООО/АО/ЗАО, оставляя только КФХ и ИП
