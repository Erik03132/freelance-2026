# Levitan Dialer Bot — инструкция

## Запуск бота

```bash
cd ~/freelance-2026/projects/levitan
source .venv/bin/activate
python3 scripts/dialer_bot.py
```

### С параметрами

```bash
# Свой extension (для второго оператора)
python3 scripts/dialer_bot.py --ext 100 --phone +79xxxxxxxxx

# Свой CSV
python3 scripts/dialer_bot.py --csv /путь/к/базе.csv

# Всё вместе
python3 scripts/dialer_bot.py --ext 100 --phone +79xxxxxxxxx --csv база.csv
```

## Загрузка базы

Три способа:
1. **Отправить файл в Telegram** — бот примет CSV, XLS/XLSX или TXT
2. **Указать при запуске** — `--csv /путь/к/файлу.csv`
3. **Без указания** — используется общая база `data/campaigns/csv/all_contacts_2026.csv`

Формат TXT: один номер на строку.  
Формат CSV: колонки `Название,Телефоны` (остальные опционально).

## Работа с ботом

В Telegram — **кнопки**, текстовые команды тоже работают:

| Кнопка | Команда текстом | Действие |
|--------|----------------|----------|
| ▶ Начать обзвон | `начать обзвон` | Запуск цикла |
| ⏭ Следующий | `следующий` | Переход к следующему контакту |
| ⏩ Пропустить | `пропустить` | Пропустить текущий |
| ⏹ Стоп | `стоп` | Остановить обзвон |
| 📊 Статус | `статус` | Статистика |

## Цикл звонка

```
1. Бот: «📞 Звоню #1: ООО АгроМир, +7900...»
2. Zoiper звонит → поднимаешь трубку
3. Слышишь гудки дозвона до клиента
4. Клиент отвечает → диалог по скрипту
5. Кладёшь трубку → жмёшь «⏭ Следующий»
6. Через ~30-90 сек: карточка CRM (расшифровка + анализ)
7. Повтор
```
