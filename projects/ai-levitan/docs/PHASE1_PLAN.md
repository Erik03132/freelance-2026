## Шаг 1: Запуск 100 звонков

**Подготовка:**
1. Выбрать CSV-базу (регион с наибольшим покрытием). Рекомендую
   `data/campaigns/csv/_for_evening_dial_2026-07-07.csv` (уже отфильтровано).
2. Проверить, что VPS доступен (для STT через Mango S2T). Если нет —
   локальный faster-whisper.
3. В Telegram боте: `▶ Начать обзвон` (или `python3 scripts/dialer_bot.py --start`).

**В процессе:**
- Бот шлёт карточку: 🏢 название, 📍 регион/район, 🏠 адрес, 🌾 культуры,
  📱 телефон, 👤 контакт (всё из базы ДО звонка).
- Оператор слышит гудки, говорит по скрипту `CALL_SCRIPT.txt`.
- После звонка: запись → STT → LLM → карточка CRM в Telegram.

**Скорость:** ~20 звонков/час (ручной режим, оператор сам жмёт «Следующий»).

## Шаг 2: Фиксация бейзлайна

После каждого звонка бот обновляет `st.stats`. В конце обзвона —
`/status` показывает сводку. Дополнительно считаем вручную/скриптом:

| Метрика | Как | Куда писать |
|---------|-----|-------------|
| % дозвона | ответили / набрано | `docs/BASELINE.md` |
| % лидов | лиды / поговорили | `docs/BASELINE.md` |
| Средняя длит. | сумма сек / кол-во | `docs/BASELINE.md` |
| Звонков/час | набрано / часы | `docs/BASELINE.md` |
| Стоимость лида | (Mango мин + LLM) / лиды | `docs/BASELINE.md` |
| Топ-5 возражений | из транскриптов | `docs/BASELINE.md` |
| % целевых культур | пшеница+ячмень / все | `docs/BASELINE.md` |

**Шаблон `docs/BASELINE.md`:**
```markdown
# Бейзлайн Фазы 1 (дата: ____)

- Всего набрано: ___
- Дозвон: ___% 
- Лиды: ___, ___% от поговоривших
- Средняя длит.: ___ сек
- Звонков/час: ___
- Стоимость лида: ___ ₽
- Топ возражений: 1. ... 2. ... 3. ...
- % целевых культур: ___%
```

## Шаг 3: Анализ топ-возражений

После 100 звонков прогнать транскрипты через LLM-сводку:
```
/summary <вставить конспект>  — для ручных
или /backfill — подтянуть из Mango S2T
```
Составить список из 10 возражений → добавить в `CALL_SCRIPT.txt`
(раздел «РАБОТА С ВОЗРАЖЕНИЯМИ») и в базу знаний Фазы 3.

## Известные ограничения

1. **Гудки** — оператор слышит 3-10 сек гудков (уберётся в Фазе 2).
2. **VPS недоступен** — STT идёт через локальный faster-whisper (медленнее,
   но работает). Mango S2T недоступен без VPS.
3. **Приветствие** — живой оператор говорит вручную (без робота, как
   решено пользователем).
