# xAI Voice Agent Builder для ai-levitan

> Дата: 06.07.2026
> Статус: Бета-версия доступна

---

## 📋 Что это

**Voice Agent Builder** — конструктор голосовых агентов без написания кода на базе Grok Voice.

| Параметр | Значение |
|----------|----------|
| Тип | No-code платформа |
| Модель | Grok Voice |
| Латентность | Sub-second |
| Номер телефона | Бесплатный (включён) |
| Создание агента | <2 минут |

---

```
Сценарий: 100 звонков × 5 мин = 500 мин

Стоимость: 500 мин × $0.05 = $25 за 100 звонков
Стоимость за звонок: $0.25

Сравнение:
```

### Шаг 2: Создать голосового агента

```python
#voice_agent.py
import asyncio
import json
import os
import websockets

async def grain_agent():
    async with websockets.connect(
        "wss://api.x.ai/v1/realtime?model=grok-voice-latest",
        additional_headers={"Authorization": f"Bearer {os.environ['XAI_API_KEY']}"}
    ) as ws:
        # Настройка агента
        await ws.send(json.dumps({
            "type": "session.update",
            "session": {
                "voice": "eve",
                "instructions": """Ты — Иван, менеджер «Глобал Филдс Экспорт».
                
Твоя задача: выяснить, есть ли у клиента зерновые к продаже.

Правила:
- Представься: «Иван, Глобал Филдс Экспорт»
- Спроси: культуру, объём, регион, цену
- Если есть объём → запиши данные
- Если нет → вежливо попрощайся
- Максимум 5 реплик

Данные для записи:
- Культура: [пшеница/ячмень/кукуруза/другое]
- Объём: [тонн]
- Регион: [область/край]
- Цена: [если назвал]
- Контакт: [телефон/whatsapp]""",
                "turn_detection": {"type": "server_vad"},
                "tools": [
                    {
                        "type": "function",
                        "name": "save_lead",
                        "description": "Сохранить лид в CRM",
                        "parameters": {
                            "type": "object",
                            "properties": {
                                "culture": {"type": "string"},
                                "volume": {"type": "string"},
                                "region": {"type": "string"},
                                "price": {"type": "string"},
                                "phone": {"type": "string"}
                            }
                        }
                    }
                ]
            }
        }))
        
        # Обработка сообщений
        async for message in ws:
            event = json.loads(message)
            if event["type"] == "response.output_audio.delta":
                # Воспроизвести аудио
                pass
            elif event["type"] == "response.function_call_arguments.done":
                # Сохранить лид
                print(f"Лид: {event}")

asyncio.run(grain_agent())
```
