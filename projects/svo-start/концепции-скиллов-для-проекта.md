## 🎯 Скилл 1: `veteran-grant-writer`

### Назначение
Помощь в подготовке заявок на гранты для ветеранского бизнеса.

### Что должен уметь:
```
- Анализировать доступные грантовые программы (региональные, федеральные)
- Генерировать структуру бизнес-плана под требования гранта
- Помогать с расчётом софинансирования (25% от суммы гранта)
- Формулировать социальную миссию (реабилитация, трудоустройство ветеранов)
- Готовить пакет документов (устав, выписка ЕГРЮЛ, смета)
- Проверять соответствие критериям (ветеранский статус, ОКВЭД)
```

### Триггеры для активации:
```
- «Подать на грант»
- «Заявка на субсидию»
- «Бизнес-план для центра Мой бизнес»
- «Соцконтракт Московская область»
- «Грант 300 тысяч ветеранам»
```

### Ожидаемый результат:
Готовый пакет документов для подачи заявки + чек-лист требований.

```

```

- «Не понимаю жену/детей»
- «Постоянная тревога»
- «Вспышки гнева»
- «Как помочь мужу адаптироваться»
- «Техники от кошмаров»

## 🔧 Технические требования к скиллам

### Структура каждого скилла:
```markdown
# Skill: {name}

## Description
Краткое описание (2-3 предложения)

## Capabilities
- Что умеет делать
- Список возможностей

## Triggers
- Слова/фразы для активации
- Когда использовать

## Workflow
1. Пошаговый процесс работы
2. Логика принятия решений

## Output Format
- Как должен выглядеть результат
- Шаблоны, чек-листы, таблицы

## Constraints
- Ограничения (что НЕ делать)
- Предупреждения о рисках

## Resources
- Ссылки на законы, источники
- Шаблоны документов
```
