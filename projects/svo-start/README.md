## 🚀 Быстрый старт для разработчика

### 1. Клонировать проект

```bash
cd /Users/igorvasin/svo-start
```

### 2. Создать скелет backend

```bash
mkdir -p backend/app/{models,routers,templates,static,utils}
cd backend
python3 -m venv venv
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install fastapi uvicorn sqlalchemy alembic pydantic python-jose passlib[bcrypt] python-multipart aiofiles jinja2 python-dotenv
```

### 4. Создать файлы

```bash
# Минимальный набор
touch app/__init__.py
touch app/main.py
touch app/config.py
touch app/database.py
touch app/models/__init__.py
touch app/models/user.py      # User, News
touch app/routers/__init__.py
touch app/routers/auth.py
touch app/routers/news.py
touch requirements.txt
touch .env
```

### 5. Запустить сервер

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 6. Проверить

```
http://localhost:8000          # Сайт
http://localhost:8000/docs     # Swagger API
```
