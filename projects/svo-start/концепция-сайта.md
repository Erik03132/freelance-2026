# Концепция веб-сайта «ООО Ветеранов СВО»

## 📖 О проекте

**Название:** ООО «Ветеранов СВО»  
**Домен:** veterany-svo.ru (пример)  
**Миссия:** Единая платформа поддержки ветеранов СВО — от юридической помощи до трудоустройства и сообщества.

---

## 🎯 Цели сайта

1. **Информирование** — гранты, льготы, поддержка ветеранов
2. **Услуги** — юристы, психологи, курсы, трудоустройство
3. **Сообщество** — форум для общения и взаимопомощи
4. **Коммерция** — магазин тематического мерча
5. **Монетизация** — устойчивая бизнес-модель для развития

## 🛠️ Техническая реализация

### Принцип:
✅ Самописное решение (Python + FastAPI)  
✅ Просто и понятно (без тяжёлых фреймворков)  
✅ Масштабируемо (от SQLite к PostgreSQL)  
✅ Экономно (нет лицензий на CMS)

### Стек:

| Слой | Технология |
|------|------------|
| Backend | Python 3.12 + FastAPI |
| База данных | SQLite → PostgreSQL |
| Frontend | Vanilla JS + Tailwind CSS + Alpine.js |
| Шаблоны | Jinja2 |
| Auth | JWT токены |
| Хостинг | Timeweb / Beget (~300 руб./мес.) |
