## 🛠️ Технический стек

| Слой | Технология | Альтернатива |
|------|------------|--------------|
| **Backend** | Python 3.12 + FastAPI | Django / Flask |
| **База** | SQLite → PostgreSQL | MySQL |
| **Frontend** | Vanilla JS + Tailwind CSS + Alpine.js | React / Vue |
| **Шаблоны** | Jinja2 | Mako |
| **Auth** | JWT | Session-based |
| **Хостинг** | Timeweb / Beget (~300 руб./мес.) | VPS |

## 🔐 Безопасность

### Реализовано:

```python
✅ HTTPS (SSL Let's Encrypt)
✅ JWT токены (7 дней)
✅ Хеширование паролей (bcrypt)
✅ Rate limiting (100 запросов/мин)
✅ CORS (настроенные origins)
✅ SQL Injection protection (SQLAlchemy ORM)
✅ XSS protection (Jinja2 autoescaping)
✅ CSRF tokens (для форм)
```
