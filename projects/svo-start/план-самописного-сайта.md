## 🛠️ Технический стек (самописный)

### Backend

| Компонент | Технология | Почему |
|-----------|------------|--------|
| Язык | **Python 3.12+** | Простой, много библиотек, быстро |
| Фреймворк | **FastAPI** | Легковесный, автодокументация, асинхронность |
| База данных | **SQLite** (старт) → **PostgreSQL** (рост) | Не требует настройки → масштабируется |
| ORM | **SQLAlchemy** | Работа с БД как с объектами |
| Шаблоны | **Jinja2** | Простые HTML-шаблоны |
| Аутентификация | **JWT токены** | Безопасно, без сессий |
| Кэш | **Redis** (опционально) | Для сессий и частых запросов |

### Frontend

| Компонент | Технология | Почему |
|-----------|------------|--------|
| HTML/CSS | **Vanilla + Tailwind CSS** | Без сборщиков, быстро |
| JavaScript | **Vanilla JS + Alpine.js** | Минимум кода, реактивность |
| Графики | **Chart.js** | Простые и красивые |
| Карты | **Leaflet** | Бесплатная альтернатива Google Maps |
| Редактор | **TinyMCE** | WYSIWYG для статей/товаров |

### Инфраструктура

| Компонент | Решение | Стоимость |
|-----------|---------|-----------|
| Хостинг | Timeweb / Beget | ~300 руб./мес. |
| Домен | .ru | ~200 руб./год |
| SSL | Let's Encrypt | Бесплатно |

### models/user.py

```python
from sqlalchemy import Column, Integer, String, Boolean, DateTime, ForeignKey, Text, Float
from sqlalchemy.orm import relationship
from datetime import datetime
import bcrypt

from app.database import Base

class User(Base):
    __tablename__ = "users"

    id = Column(Integer, primary_key=True)
    email = Column(String, unique=True, index=True, nullable=False)
    password_hash = Column(String, nullable=False)
    role = Column(String, default="user")  # user, admin, moderator
    
    first_name = Column(String)
    last_name = Column(String)
    phone = Column(String)
    
    is_veteran = Column(Boolean, default=False)
    veteran_status = Column(String)  # participant, veteran, family
    
    created_at = Column(DateTime, default=datetime.utcnow)
    updated_at = Column(DateTime, onupdate=datetime.utcnow)
    
    # Связи
    appointments = relationship("Appointment", back_populates="user")
    orders = relationship("Order", back_populates="user")
    forum_topics = relationship("ForumTopic", back_populates="user")
    forum_posts = relationship("ForumPost", back_populates="user")
    
    def set_password(self, password: str):
        self.password_hash = bcrypt.hashpw(
            password.encode(), bcrypt.gensalt()
        ).decode()
    
    def check_password(self, password: str) -> bool:
        return bcrypt.checkpw(
            password.encode(), self.password_hash.encode()
        )


class News(Base):
    __tablename__ = "news"
    
    id = Column(Integer, primary_key=True)
    title = Column(String(200), nullable=False)
    slug = Column(String(200), unique=True, index=True, nullable=False)
    content = Column(Text, nullable=False)
    
    category = Column(String(50), index=True)  # legislative, events, grants, psychology, courses, shop, partners
    author_id = Column(Integer, ForeignKey("users.id"))
    thumbnail = Column(String(500))  # URL изображения
    
    views = Column(Integer, default=0)
    is_important = Column(Boolean, default=False)  # Важная новость (красная метка)
    is_published = Column(Boolean, default=False)
    published_at = Column(DateTime)
    created_at = Column(DateTime, default=datetime.utcnow)
    
    # Связи
    author = relationship("User", backref="news")
```

### routers/auth.py

```python
from fastapi import APIRouter, HTTPException, Depends
from pydantic import BaseModel, EmailStr
from sqlalchemy.orm import Session
from datetime import datetime, timedelta
import jwt

from app.database import get_db
from app.models.user import User

router = APIRouter()

SECRET_KEY = "your-secret-key-change-in-production"
ALGORITHM = "HS256"

class RegisterRequest(BaseModel):
    email: EmailStr
    password: str
    first_name: str
    last_name: str

class LoginRequest(BaseModel):
    email: EmailStr
    password: str

class TokenResponse(BaseModel):
    access_token: str
    token_type: str = "bearer"

def create_token(user_id: int) -> str:
    expire = datetime.utcnow() + timedelta(days=7)
    return jwt.encode(
        {"sub": user_id, "exp": expire},
        SECRET_KEY,
        algorithm=ALGORITHM
    )

@router.post("/register", response_model=TokenResponse)
async def register(data: RegisterRequest, db: Session = Depends(get_db)):
    # Проверка: есть ли уже
    existing = db.query(User).filter(User.email == data.email).first()
    if existing:
        raise HTTPException(400, "Email уже зарегистрирован")
    
    # Создаём
    user = User(
        email=data.email,
        first_name=data.first_name,
        last_name=data.last_name
    )
    user.set_password(data.password)
    
    db.add(user)
    db.commit()
    db.refresh(user)
    
    return TokenResponse(access_token=create_token(user.id))

@router.post("/login", response_model=TokenResponse)
async def login(data: LoginRequest, db: Session = Depends(get_db)):
    user = db.query(User).filter(User.email == data.email).first()
    if not user or not user.check_password(data.password):
        raise HTTPException(401, "Неверный email или пароль")
    
    return TokenResponse(access_token=create_token(user.id))
```

## 🚀 Запуск проекта

### 1. Установка зависимостей

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Настройка окружения

```bash
# .env
DATABASE_URL=sqlite:///./app.db
SECRET_KEY=your-secret-key-change-in-production
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=587
SMTP_USER=info@veterany-svo.ru
SMTP_PASS=your-password
```

### 3. Запуск сервера

```bash
# Development
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# Production
uvicorn app.main:app --host 0.0.0.0 --port 8000 --workers 4
```

### 4. Проверка

```
http://localhost:8000          # Сайт
http://localhost:8000/docs     # API документация (Swagger)
http://localhost:8000/redoc    # API документация (ReDoc)
```
