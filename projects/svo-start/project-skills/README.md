## 📖 Как использовать

### Для Claude Code / Gemini CLI / Cursor

1. **Скопируйте папку `skills`** в ваш проект или домашнюю директорию
2. **Активируйте нужный скилл** через команду или контекст:
   ```
   Используй скилл veteran-grant-writer для подготовки заявки на грант
   ```
3. **AI применит инструкции** из соответствующего SKILL.md

### Примеры использования

```
# Гранты
«Используй veteran-grant-writer: подготовить заявку на грант 300 тыс. для ООО ветеранов СВО»

# Юридические вопросы
«Используй military-legal-consultant: какие льготы по налогу на имущество у ветерана СВО»

# Психология
«Используй adaptation-psychologist: техники от кошмаров после возвращения»

# Курсы
«Используй military-course-designer: создать структуру курса по стрессоустойчивости»

# Мерч
«Используй military-merch-manager: поставщик шевронов для подразделения 50 штук»

# Бизнес
«Используй veteran-business-architect: точка безубыточности для ООО с 3 сотрудниками»

# Контент
«Используй seo-veteran-content: статья про гранты ветеранам СВО 2026»

# Соответствие
«Используй compliance-military-law: проверить устав ООО на соответствие гранту»
```

## 🏗️ Структура скилла

Каждый скилл содержит:

```yaml
---
name: skill-name
description: Краткое описание
metadata:
  version: 1.0.0
  region: Russia
  target: Целевая аудитория
---

## Use this skill when
- Когда использовать скилл

## Do not use this skill when
- Когда НЕ использовать

## Instructions
- Общие инструкции для AI

## Expert Purpose
- Описание роли и экспертизы

## Capabilities
- Список возможностей скилла

## Behavioral Traits
- Поведенческие характеристики

## Knowledge Base
- База знаний (законы, источники)

## Response Approach
- Пошаговый подход к ответам

## Example Interactions
- Примеры запросов от пользователей

## Resources
- Ссылки на дополнительные ресурсы

## Red Flags
- Предупреждения о рисках

## Constraints
- Ограничения и запреты

## Remember
- Ключевые принципы
```

## 🔄 Обновление скиллов

Скиллы требуют регулярного обновления:

1. **Проверяйте законы** — законодательство меняется часто
2. **Обновляйте ресурсы** — контакты, сроки, суммы грантов
3. **Собирайте фидбэк** — от пользователей и AI
4. **Версионируйте** — указывайте версию в metadata

## 🤝 Вклад в коллекцию

Для добавления нового скилла:

1. Создайте папку `skills/your-skill-name/`
2. Добавьте файл `SKILL.md` по структуре выше
3. Протестируйте с AI-ассистентом
4. Обновите этот README
