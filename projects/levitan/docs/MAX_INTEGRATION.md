# Интеграция MAX мессенджера в проект Levitan

## Что такое MAX

**MAX** — российский национальный мессенджер (ранее «Максим»). Аналог Telegram с акцентом на конфиденциальность и государственную поддержку.

- **Сайт**: https://max.ru
- **Веб-версия**: https://web.max.ru
- **Скачивание**: Google Play, RuStore, AppGallery

## Зачем использовать MAX в проекте

| Преимущество | Описание |
|--------------|----------|
| **Российский сервер** | Данные хранятся в РФ, нет блокировок |
| **Бесплатные звонки** | Внутренние звонки через мессенджер |
| **Каналы** | Создание каналов для рассылки |
| **Господдержка** | Интеграция с Госуслугами |

## Техническая реализация

### Вариант 1: Ручное приглашение

Оператор при звонке диктует ссылку на канал.

### Вариант 2: Автоматическая SMS

После звонка бот отправляет SMS со ссылкой (требуется согласие).

### Вариант 3: WhatsApp рассылка

После согласия клиента — отправка шаблонного сообщения через WhatsApp Business API.

## Интеграция с текущим проектом

### Текущий пайплайн

```
Звонок → Запись → STT → LLM → CRM → [НОВОЕ: MAX канал]
```

### Дополнительные шаги

1. **При звонке**: оператор упоминает канал MAX
2. **После звонка**: если клиент согласен → SMS со ссылкой
3. **В CRM**: пометка «Подписан на MAX канал»
4. **В канале**: публикация актуальных цен и условий
