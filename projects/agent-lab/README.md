# 🧬 Agent Lab — Лаборатория AI-Агентов

> Core Agent — шаблон. Анжела — его ребёнок.

## Идея

Один **универсальный агент** (ядро) с современным стеком:
- ✅ ReAct цикл (сам решает, какие инструменты вызвать)
- ✅ MCP (стандартные серверы инструментов)
- ✅ RAG (память / база знаний)
- ✅ Tool Calling (структурированный вызов функций)

Чтобы получить **конкретного специалиста**, подключаем:
- 📚 Специализированные знания (RAG-массив)
- 🎭 Системный промпт (характер, манера, правила)
- 🔧 Специфичные MCP-серверы (Bitrix, Avito, GitHub...)

## Примеры "детей"

| Ребёнок | Знания | Характер | MCP-серверы |
|---------|--------|----------|-------------|
| **Анжела** | Птицеводство, цены, доставка | Продавец, 4 роли | Bitrix, Avito |
| **Шерлок** | Рынок, конкуренты, SEO | Аналитик | Web Search, Google |
| **Мустай** | Законодательство РБ | Сенатор-помощник | Парламентские API |
| **Freelance Agent** | Биржи, портфолио | Менеджер проектов | Kwork, FL.ru |

## Быстрый старт

```bash
cd agent-lab
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # заполнить ключи
python core_agent.py
```

## Стек

- **Python 3.11+**
- **LangGraph** — оркестрация ReAct-цикла
- **FastMCP** — MCP-серверы инструментов
- **ChromaDB** — локальная векторная БД (для лаборатории)
- **LangChain** — абстракции над LLM и инструментами
- **OpenRouter / Gemini** — LLM провайдеры
