# Навык: SVG to 3D Workflows (Рембрандт)

## 📌 Описание навыка
Рембрандт интегрировал в свой арсенал процессы генерации и обработки векторной графики для вывода в трехмерное пространство (WebGL).

Как AI-агент, Рембрандт не может "нажать на кнопку мыши" в 3D редакторе, но **в совершенстве владеет кодом генерации векторов (SVG)**, который является математической основой для последующей 3D-экструзии.

## 🚀 Два пути реализации:

### Путь 1. Внешний рендеринг (через 3dsvg.design или Spline)
**Сценарий:** Быстрая подготовка 3D-ассетов для соцсетей, баннеров или видео-презентаций.
1. Пользователь запрашивает у Рембрандта: *"Создай логотип/иконку в формате SVG".*
2. Рембрандт генерирует чистый, геометрически точный `<svg>` код с `<path>`, замкнутыми контурами и без `<g>` артефактов.
3. Пользователь сохраняет код в `.svg` файл и переносит в `3dsvg.design` (или Spline) для настройки теней, PBR-материалов, света и рендера в 4K.
**Преимущество:** Экономия десятков часов на кодинге шейдеров и света.

### Путь 2. Интерактивный WebGL-код (In-house Engine)
**Сценарий:** Интерактивное 3D на сайте клиента (наведение мышкой, скроллинг), когда картинки/видео недостаточно.
1. Рембрандт пишет React-компонент с использованием `@react-three/fiber` и `@react-three/drei`.
2. Используется `three/addons/loaders/SVGLoader.js` для парсинга SVG-контуров налету.
3. Добавляется `ExtrudeGeometry` для "выдавливания" путей в полноценный 3D-mesh.
4. Добавляются PBR-материалы (`MeshPhysicalMaterial`), освещение (`Environment`, `SpotLight`) и физика (`PresentationControls`).
**Преимущество:** 100% кастомизация, полный контроль над моделью прямо внутри JS/TS-кода сайта.
