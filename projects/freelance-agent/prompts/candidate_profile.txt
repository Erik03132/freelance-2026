Роль: AI-инженер / промпт-инженер / архитектор AI-процессов; автоматизация рутины и бизнес-процессов.

Стек: Python (уверенно), LLM API (Claude / ChatGPT / Gemini), агентные шины (собственная A2A — функционально как LangGraph), RAG / embeddings / vector search / reranking (как LangChain), observability-слой (как Langfuse: трассы, метрики), pm2 / cron, Docker, SQL / NoSQL, веб-интерфейсы (FastAPI / Flask + фронт), Telegram-боты (aiogram / python-telegram-bot). Claude Code / Cursor — осваивает под стандарты команды.

Интеграции: CRM (Bitrix24), телефония (Mango), площадки Avito, VK и другие (маркетплейсы и соцсети перечислять раздельно, не объединять в один «маркетплейс»), Google Таблицы; 1С / «Мой склад» — по тому же паттерну API (готов взять в работу).

Кейсы:
- Многоагентная A2A-шина: реестр агентов, диспетчер, governance (HMAC-подпись, скан инъекций), observability; 5 агентов работают как единый конвейер, оркестратор запускает рутины и шлёт сводки.
- Bitrix-сканер забытых сделок: Python + REST API по расписанию, правила «если статус X и дней > N», классификация застоя, сводка.
- Голосовой обзвон Levitan (Mango + AI + CRM): кампания на 100+ звонков, метрики воронки, обогащение карточек.
- RAG / семантический поиск: embeddings, vector search, reranking; замеры качества (Hit@k, MRR-подобные) на эталонных парах.
- Генерация и GEO-оптимизация контента: конвейер текстов / постеров, распределение (VK, Avito), рост видимости в выдаче ИИ.
- Ежедневные аналитические сводки и дашборды; библиотеки промптов и регламентов (CONTEXT, ADR).

Метрики (proof bank): −69% расхода токенов при росте качества; 5 агентов как единый конвейер; автоматизация вошла в регулярное использование (ежедневные сводки, сканирование сделок).

Доп. опыт: руководство группой 5–15 человек 2+ года; контекст EdTech.
Дизайн/макеты по слоям: Google AI Studio и Stitch from Google — генерация UI-макетов и прототипов (сильнее чистой Figma). Figma как редактор — не использовал (зона роста, беру быстро).
Инструменты ИИ в работе — обязательно (Claude / ChatGPT / Gemini для прототипирования и промпт-инжиниринга).
