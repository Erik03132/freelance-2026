## ОТЧЁТЫ — ДВА ОТДЕЛЬНЫХ, ПО ЗАПРОСУ

Отчёты генерируются **по запросу** пользователя в чате с агентом.
Два отчёта — **разные скрипты, разные данные, разные CRM**.

### 1. Отчёт Заботкиной (CRM — боевой Битрикс)

**Триггер:** Пользователь пишет:
- «отчёт Заботкиной за сегодня»
- «отчёт Заботкиной за 28 апреля»
- «представь отчёт Заботкиной за последние 3 дня»

**Что делает агент:**
1. Запускает `bitrix_scanner.py` → свежий скан ОСНОВНОГО Битрикса
2. Запускает `daily_report.py` или `manual_report.py` → генерация отчёта
3. Показывает результат пользователю в чате

**Данные:** Сделки, звонки, менеджеры, конверсии, лиды, воронка
**CRM:** ОСНОВНОЙ Битрикс (`BITRIX_WEBHOOK_URL`)
**Скрипты:** `bitrix_scanner.py` → `daily_report.py`

### 2. Отчёт Птенчиковой (Проект — Песочница Битрикс)

**Триггер:** Пользователь пишет:
- «отчёт Птенчиковой за сегодня»
- «представь отчёт Птенчиковой за последние 3 дня»

**Что делает агент:**
1. Запускает `sandbox_scanner.py` → скан ПЕСОЧНИЦЫ
2. Формирует отчёт по задачам, прогрессу, маркетингу
3. Показывает результат пользователю в чате

**Данные:** Задачи проекта, маркетинг, прогресс, планы
**CRM:** ПЕСОЧНИЦА (`SANDBOX_BITRIX_WEBHOOK_URL`)
**Скрипты:** `sandbox_scanner.py` → (нужен новый скрипт отчёта)

## 🤫 HEARTBEAT RULES — ПРАВИЛА МОЛЧАНИЯ (02.05.2026)

> Вдохновлено: OpenClaw HEARTBEAT.md
> Реализация: `agent/heartbeat_rules.py`

### Золотое правило
**Молчи когда всё ок. Пиши ТОЛЬКО когда нужно внимание.**

### Тихие часы
- **22:00 — 07:00 MSK** — не беспокоить
- Исключение: CRITICAL (сайт/CRM упал) — пишем ВСЕГДА

### Уровни серьёзности

| Уровень | Когда | Ночью (22-07) | Днём (07-22) |
|---------|-------|---------------|--------------|
| **CRITICAL** | Сайт упал, данные потеряны | ✅ ШЛЁМ | ✅ ШЛЁМ |
| **WARNING** | Процесс упал но вылечен | 🤫 молчим | ✅ ШЛЁМ |
| **INFO** | Отчёт, дайджест | 🤫 молчим | ✅ ШЛЁМ |
| **SILENT** | Всё ок, heartbeat alive | 🤫 ВСЕГДА | 🤫 ВСЕГДА |

### Кому отправлять
- **Андрей (ADMIN)** — ТОЛЬКО готовый ежедневный отчёт (по запросу)
- **Игорь (OWNER)** — алерты, отчёт по запросу
- **НИКОГДА автоматически**: алерты ночью, мусорные отчёты, health monitor спам
