# 🪖 Предложение: «Армия Поддержки» — легальный прогрев сообществ

> Статус: 💡 ИДЕЯ — на ближайшее будущее  
> Дата: 03.05.2026  
> Автор: Antigravity AI

---

## Концепция

Вместо ботов — **реальные амбассадоры** (живые люди), которые за небольшое вознаграждение
выполняют целевые действия в группах VK/OK:
- вступают в группу
- ставят лайки
- оставляют живые комментарии по скриптам
- делают репосты

Цель — создать эффект «живого» сообщества на старте, запустить алгоритмический маховик.

---

## Механика

### Фаза 1: Вербовка (бесплатно)
- База клиентов из Bitrix24/Avito
- Друзья/знакомые
- Локальные чаты Telegram
- Студенческие группы ВКонтакте

### Фаза 2: Задания и бонусы
| Действие | Бонус |
|---|---|
| Вступить в группу | 5 ₽ или скидка |
| Лайки на 3 поста | +5 ₽ |
| Комментарий (живой) | +10 ₽ |
| Репост в свою страницу | +20 ₽ |
| Привести друга | +50 ₽ |

**Бюджет:** ~100 ₽/человек = 3 000 ₽ за 30 активных людей

### Фаза 3: Автоматизация учёта (Telegram-бот)
```
/start → инструкция с заданиями
/done → скриншот выполнения → запись в БД
→ ты проверяешь → начисляешь бонус
```

### Фаза 4: Скрипты для комментариев
Заготовленные живые фразы:
- "Скажите, а когда следующая поставка?"
- "Забирал вчера — всё отлично, спасибо!"
- "А есть скидки на оптовые заказы?"

---

## Стек реализации
- Python + aiogram (Telegram-бот учёта)
- SQLite (база задач и выполнений)
- Интеграция в дашборд (статус кампаний)
